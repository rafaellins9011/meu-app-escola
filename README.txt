# Busca Ativa Escolar - Firebase Hosting

## COMO PUBLICAR

1. Extraia/crie esta pasta em seu computador.
2. No terminal, entre nela e rode:

firebase login
firebase init

> Escolha "Hosting"
> Escolha o projeto: busca-ativa-escolar
> Use "build" como pasta pública
> Sim para SPA (Single Page App)
> Não para sobrescrever index.html

3. Depois, execute:

firebase deploy

✅ Seu painel estará no ar em: https://busca-ativa-escolar.web.app
✅ O que fazer agora:
Crie a pasta busca_ativa_escolar/ no seu computador

Cole esses conteúdos em arquivos .html e .json como descrito

Abra o terminal e rode:

cd caminho/para/busca_ativa_escolar
firebase login
firebase init
firebase deploy

Se quiser, posso te acompanhar passo a passo aqui ao vivo.
